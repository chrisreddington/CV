Cloud With Chris

Cloud with Chris is Chris' personal blog, podcast and content creation platform. Chris regularly invites guests to talk about their experiences with the cloud and hear about some of the lessons learned around their cloud journey

Content typically focuses on Azure, Azure DevOps, GitHub, and other cloud related topics.

Cloud With Chris is a JAMStack application deployed on Azure Static Web Apps behind Azure Front Door. It is built on top of the Hugo static site generator, and uses a custom theme (Chris' open sourced Hugo theme for content creators) to render the content. The theme has been designed fully responsive, with accessibility and SEO in mind.

Hugo Creator

Born through Chris' desire to create content on Cloud With Chris, Hugo Creator is a Hugo theme designed for Content Creators.

Released as Open Source under the MIT license, it is a fully responsive theme designed to be used with Hugo. It has been built with accessibility and SEO in mind, and has Progressive Web App support for offline viewing.
